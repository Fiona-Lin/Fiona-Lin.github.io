My dual boot story

12-March-2016

This story was begin last July, I just got an offer to Usyd. So I my study gears--asus zenbook. However long story short, I got asus laptop which is similar model to my budy's and he showed me that he dual boot Mac ox and window 10 in his laptop when we catch up in the school holiday, so I asked him to do the same in mine. In the end, because my model is different than his, also we kinda hack the Mac os into an non-mac machine. Therefore Mac ox and window crash one or the other whichever boot first.

I had to give up in the end. My budy has my laptop reverse to the origin setting and I also updated my HHD to SSD for performance reason.

After a couple month, I self-learnt a bit python and did a little bit python in linux. Also I knew that I will use linux in my next semester study so I try to install ubuntu along with the existing window. I did a bit research about the dual boot ubuntu and window however it was not as easy as I read.

    Problems I encountered:
  1. I download ubuntu 15.04, which is not really UEFT supported version. I didn't know this until I install the ubuntu and it does not work and so I got to use the legacy mode
  2. My model asus laptop comes with window 8 and UEFT using GPT hard disk partition format type. Because the ubuntu is not support UEFT itself and it is in legacy model.

Therefore, we found a way to convert GPT to MBR and complete the boot repair. In the end, I achieved the dual boot 2 systems in a legacy mode.
